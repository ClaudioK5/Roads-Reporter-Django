# Веб-приложение для обработки данных об автомобильных дорогах

Данное приложение разработано на Django и предназначено для загрузки Excel-файла с данными об автомобильных дорогах, обработки этих данных и автоматической генерации отчета в формате Word.

## Функциональность приложения

Приложение позволяет:

- загрузить Excel-файл с данными об автомобильных дорогах
- проверить корректность структуры Excel-файла
- обработать данные с использованием библиотеки Pandas
- вычислить общую протяженность всех дорог
- автоматически сформировать отчет в формате Word (.docx)
- скачать готовый отчет

## Требования к Excel-файлу

Excel-файл должен содержать следующие колонки в указанном порядке:

№ п/п  
Наименование  
Значение автомобильной дороги  
Категория  
Протяженность, км  

Приложение проверяет структуру файла перед обработкой данных.

## Как запустить проект локально

1. Склонировать репозиторий:


git clone https://github.com/ClaudioK5/Roads-Reporter-Django.git


2. Перейти в папку проекта:


cd Roads-Reporter-Django


3. Установить зависимости:


pip install -r requirements.txt


4. Выполнить миграции базы данных:


python manage.py migrate


5. Запустить сервер разработки:


python manage.py runserver


После этого приложение будет доступно по адресу:


http://127.0.0.1:8000


---

## Как использовать приложение

1. Открыть веб-страницу приложения  
2. Выбрать Excel-файл с данными об автомобильных дорогах  
3. Нажать кнопку **"Загрузить"**  
4. После обработки данных появится сообщение с общей протяженностью дорог  
5. Скачать автоматически сгенерированный Word-отчет
